# Contributor Covenant Code of Conduct

## Our pledge

We as members, contributors, and maintainers pledge to make participation in the Lattice Current community a harassment-free experience for everyone.

## Standards

Examples of expected behavior:

- constructive feedback
- respectful disagreement
- clear communication
- accountability for mistakes

Examples of unacceptable behavior:

- harassment
- abuse
- discrimination
- publishing private information without permission

## Scope

This code of conduct applies to repository spaces and associated community interactions.

## Reporting

Report unacceptable behavior through the repository maintainer or issue-management channels used by this repository.
