# 2026-03: 앱 수준 글로브 쇼케이스 추가

## 무엇이 바뀌었나

- `showcase/globe` 전용 페이지를 추가했습니다.
- 홈의 가벼운 데모와 별도로 더 무거운 읽기 전용 글로브를 제공하게 됐습니다.
- 텍스처 지구, 밤하늘 배경, 관계 아크, 무역 경로, 케이블 경로, 전장 상세 패널이 들어갔습니다.

## 왜 중요한가

라이브 피드나 private runtime 없이도 실제 앱에 더 가까운 지도 시각화를 문서 사이트에서 보여줄 수 있게 됐습니다.

## 사용자 영향

- 실제 앱을 열기 전 더 깊은 지도 표면을 확인할 수 있습니다.
- 이전의 단순 홈 데모보다 제품의 시각 밀도를 더 잘 전달합니다.
