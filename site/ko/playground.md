# 플레이그라운드

<p class="lc-section-caption">
이 페이지는 실제로 눌러보는 체험용입니다. 아래 모든 인터페이스는 가상 데이터로 동작하므로 라이브 백엔드 없이도 제품 흐름을 이해할 수 있습니다.
</p>

## 이 페이지에서 볼 수 있는 것

- 실시간 이벤트 선택과 운영 요약
- 이벤트 -> 테마 -> 자산 전이 확인
- 리플레이 스타일 의사결정 검토
- 매크로 슬라이더 기반 가정 시나리오

## 제한

- 가상 데이터만 사용
- 외부 API 없음
- private feed, secret, 서비스 endpoint 노출 없음
