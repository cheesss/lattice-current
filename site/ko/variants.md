# 변형

Lattice Current는 하나의 저장소에서 여러 제품 표면을 제공합니다.

| 변형 | 초점 | 대표 사용자 |
| --- | --- | --- |
| `full` | 지정학, 분쟁, 인프라, 인텔리전스 | 애널리스트, 운영자, OSINT 모니터 |
| `tech` | AI, 클라우드, 스타트업, 사이버, 생태계 매핑 | 기술 전략, 벤처, 플랫폼 리서치 |
| `finance` | 매크로, 크로스애셋, 시장 전이, 리플레이, 아이디어 | 매크로 및 이벤트 드리븐 리서치 |

## 공통 기반

모든 변형은 다음 코어를 공유합니다.

- 데이터 수집과 정규화
- AI 보조 요약과 Q&A
- 온톨로지와 그래프 서비스
- 이벤트 전이 모델링
- 리플레이와 백테스트 프리미티브

## 문서 범위
