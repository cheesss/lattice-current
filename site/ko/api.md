# API

Lattice Current는 여러 typed service domain을 노출합니다. 이 저장소의 공개 API 문서는 OpenAPI 아티팩트로 관리됩니다.

## 주요 레퍼런스

- [API 문서 디렉터리](https://github.com/cheesss/lattice-current/tree/main/docs/api)
- [엔드포인트 추가 가이드](https://github.com/cheesss/lattice-current/blob/main/docs/ADDING_ENDPOINTS.md)

## 서비스 패밀리

- News
- Market
- Conflict
- Maritime
- Cyber
- Climate
- Displacement
- Research
- Intelligence

## 메모

문서 사이트는 저장소 레퍼런스 문서로 들어가기 위한 내비게이션 레이어입니다. 더 깊은 스키마 세부는 생성된 OpenAPI 파일에 남아 있습니다.
