# ライブインテリジェンス

## 何をするか

ライブフィード、マップレイヤー、スコア、AI サマリーを1つの運用画面に統合します。

## なぜ必要か

ニュースフィード、地図、別々の市場ツールの間を行き来するコンテキスト切り替えを減らすためです。

## 入力

- 厳選されたフィードと API
- 地図レイヤーと地理空間アセット
- source credibility と signal aggregation の出力
- AI サマリーと ontology context

## 出力

- ライブパネルと地図オーバーレイ
- alert カードと instability score
- focal point と transmission lead

## 主な UI 面

- 地図とレイヤー制御
- ライブニュースパネル
- Analysis Hub と Ontology ページ
- 戦略レベルおよび国別サマリー

## 関連アルゴリズム

- signal aggregation
- source credibility
- convergence と instability scoring
- ontology graph enrichment

## 制限

公開ドキュメントでは機微な運用コネクタや非公開ソースは公開しません。

## バリアント適用範囲

`full`、`tech`、`finance` に共通し、ドメイン別にフィードとパネルが変わります。
