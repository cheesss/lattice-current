# アルゴリズム

Lattice Current は、決定論的スコア、online prior、グラフロジック、リプレイベース評価を組み合わせています。

## 主なグループ

- source credibility と truth scoring
- country instability と convergence scoring
- event-to-market transmission
- regime-aware weighting
- ontology 制約と graph inference
- replay と walk-forward backtesting
