# UX and Visualization Improvements

## Summary

World Monitor benefits most when the interface reduces search friction and promotes decision-ready context.

## High-impact additions

1. Context-aware focus mode across news, maps, investment, and backtesting.
2. Timeline minimap for fast jumps into risk spikes.
3. Transmission network view alongside sankey for causal exploration.
4. Regime dial and uncertainty visuals for investment interpretation.
5. Equity curve and decision heatmap for backtest readability.
6. Hover evidence previews for fast source inspection.
7. Explainability badge showing data-driven versus inference-heavy outputs.
8. Workspace presets for command-center, deep-dive, tech, and finance workflows.
9. Import/replay job queue with progress, retry, and last-error detail.
10. Run comparison view for backtest configurations.
11. What-if sandbox sliders for macro shock simulation.
12. Human-in-the-loop false-positive review deck.
13. Inline backlinks from AI-generated claims to evidence cards.
14. Better mobile focus routing for key panels.
15. Stronger urgency animation for critical alerts without overloading the UI.

## Recommended build order

1. shared context and focus routing
2. job queue and replay ergonomics
3. evidence hover and explainability badge
4. what-if sandbox
5. analyst review tooling
