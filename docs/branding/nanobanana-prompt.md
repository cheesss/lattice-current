# Nanobanana Prompt

## GitHub Hero / Social Preview Prompt

Create a premium GitHub hero image for an open-source project called "World Monitor".

World Monitor is a real-time geopolitical and market intelligence platform that combines global OSINT, live news monitoring, AI-assisted analysis, ontology graphs, event-to-market transmission modeling, historical replay, and backtesting.

Visual style:
A cinematic but credible intelligence command center, not sci-fi fantasy.
Show a large world map as the visual anchor, with glowing regions of geopolitical tension, subtle route arcs, H3-style intelligence density cells, threat clusters, and refined analytical overlays.
Around the map, integrate elegant panels or translucent UI fragments suggesting:
- live breaking news
- country instability scores
- event-to-asset transmission network
- ontology and knowledge graph
- equity curve and backtest analytics
- AI reasoning cards
- regime gauge and risk dashboard

Mood:
institutional-grade, high-trust, strategic, analytical, sharp, modern, disciplined.
Avoid cartoonish hacker aesthetics. Avoid generic neon cyberpunk. Avoid unreadable tiny text.

Color palette:
deep navy, graphite, steel, muted cyan, amber, restrained red alerts.

Composition:
clean center-weighted layout with enough negative space for GitHub header crop.
If text appears, keep it minimal and legible: "World Monitor".

Aspect ratio:
16:9.
